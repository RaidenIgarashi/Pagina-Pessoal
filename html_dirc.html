Dicionário

Nome Função
html É a raiz do HTML, todo o conteúdo da página vai dentro dele
head é onde vai os meta dados do documento
body É o corpo do HTML. É aqui onde você coloca todas as coisas que vão aparecer no seu site
main Serve para conter o conteúdo principal da pagina
nav É usado para representar a barra de navegação, geralmente a que fica em cima das páginas
div O contêiner mais básico, sem sentido semântico. Usado apenas para juntar elementos e aplicar css
section Um contêiner quer agrupa elementos que se relacionam para formar, por exemplo, um tópico
header É um contêiner que serve como cabeçalho da página, onde vai o título(h1), logotipos, menus e etc
footer É um contêiner que serve como rodapé, aqui geralmente se coloca os link de contato e redes sociais
h1 Título principal da página, como o logo
h2 Titulo secundário, como título de um tópico
h3 Subtitulo de terceiro nível
p Parágrafo de texto
ul Lista não ordenada com marcadores(as bolinhas)
li Elementos da lista
a Link(ancora) usado para redirecionar o usuário para outro página, tópicos ou sites externos
img Usado para colocar imagens
adress Usado para informar informações do dono do site, como email, telefone de contato, rede social e etc
link é usado dentro da tag head, é usado para conenctar o html com recursos externos, como o CSS
table Cria uma tabela
tr Cria uma linha na tabela
th é usado para o cabeçalho da tabela. o Texto vem centralizado e em negrito
td Célula comum da tabela
